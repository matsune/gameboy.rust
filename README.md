# gameboy.rust
A Rust implementation of Gameboy Emulator

## Features
- Supporting various cartridge types
  - ROM Only
  - MBC1
  - MBC2
  - MBC3
  - MBC5
- Save data to file
- Sound on/off
- GBC roms
